# Choice
A user is presented with a list of choices. They can select none or n

## Presentation
How a choice can be presented

### Dropdown
Includes:
- Select
- Combobox


### List
Includes:
- Radio group (with or without card borders)
- Checkbox group (with or without card borders)
- Listbox (inline with possible scroll)
- Segmented control (when 1 only allowed)
- Tree view (nested choices)
- Table row selection (single or checkbox)
